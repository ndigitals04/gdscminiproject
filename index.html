Introduction

HTML- Hyper Text Transfer Protocol. It is a markup language that composes the structure of any website. Its language is made up of elements that define how it's content should be displayed. Anyone looking at getting into web design or development would need to have an understanding of HTMl.

Prerequisites

Few things you'll need

  • A code editor (VS code, Sublime Text, Atom)
  • A browser (Chrome,Firefox,Edge)
  • Your own focus

Functionality

Its good to note that HTML is only a markup language and not a programming language.Html code is written in a code editor and processed in a browser to show it's contents. Html code are written in pairs of tags, an opening and a closing one. "<h1> Heading </h1>" The <h1> tag represents a header and the text between both tags would be displayed in the browser as a header

HTMl Basics

  • HTML documents start with a "<!Doctype html declaration>" decla ring that the page is in HTML5.
  • The actual code starts with the "<html>" tag
  • It continues with the "<head>" tag which contains meta tags
  • The "<title>" tag is used to display the title of the page It is placed inside the "<head>" tag
  • The "<body>" tag contains the visible content of the page
  • Always remember to close your tags"</body>"(Note that not all tags have closing tags)

HTML elements

  • <h1> -<h6>- Defines HTML headings in order of their importance
  • <p> Defines HTML paragraphs
  • <a> Defines HTML hyperlinks
  • <ul> Defines HTML Unordered Lists
  • <ol> Defines HTML ordered lists
  • <li> Defines the listed item
  • <!-- --> Defines HTML Comments so that others can see understand what you're trying to do
  • <em> Defines Emphasized text
  • <strong> Defines Important Text
  • <b> Defines bold text
  • <i> Defines Text in italics

HTML Attributes

HTML attributes are used on elements to convey extra meaning or contain extra details of that element.

  • lang attribute- It is used to specify the language that the webpage is written <html lang="en"> This specifies the language in English. This enables Web browsers and Search engine to give better search results to their users.
  • style attribute- The style attribute is used on HTML elements so that CSS styles can be applied to it "<h1> style="color:white; background-color: blue;">N-Digitals</h1>" This specifies the color and background color of the content of the h1 element 'N-Digitals'
  • src attribute- The src attribute is used in the img element to specify a path to the location of the image to be displayed on the webpage "<img src= "./images/ndigitals-logo" />" This specifies the location (url) of the ndigitals-logo image
  • alt attribute- This is used in img elements to define alternative text to be displayed in case of error in display of the original image. It is also good for screen readers to be able to know what is on the page "<img src= "./images/ndigitals-logo" alt= "ndigitals logo" />"
  • href attribute This attribute specifies the URL to somewhere else on the same webpage, or another webpage of the same website or another website entirely. The path specified can either be a relative URL or an absolute URL. "<a href= "./ndigitals_projects/cat-ai.html"> N-Digitals Projects</a> This is relative URL because the folders are within the same computer. <a href= "https://ndigitals04.github.io/portfolio-website"> N-Digitals Portfolio Website</a>" The path specified is an absolute URL cause it's located on another PC.
  • id attribute- The id attribute is used to give a unique identity to an element so as to be styled in css or to be referenced in JavaScript. <h1 id="heading">N-Digitals</h1>
  • class attribute- The class attribute is just like the id attribute except that it can be used for multiple elements. It's very useful when you want to style a particular set of elements in a certain way <p class= "paragraph">Our services</p>
  • title attribute This is used to give more information about the content of an element. <p title="We offer services such as catering, baking, cooking"> Our services include</p> This is that stuff that appears when you hover over some text in a webpage
